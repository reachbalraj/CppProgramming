
copy existing git Repo:               git clone https://github.com/reachbalraj/CppProgramming.git
download changes from repo :          git fetch origin
update local head to latest changes:  git pull origin master
current status of local repo :        git status
add files to staging area :           git add/ git add .
commit the changes :                  git commit -m "text message describes the changes"
push the changes to remote repo:      git push
